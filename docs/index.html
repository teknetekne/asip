1. Discovery

Nodes connect to the Hyperswarm DHT using a shared topic hash. They find each other without IP addresses or DNS.

2. Verification

Every message is signed with the sender's private key. Nodes verify signatures before executing tasks to prevent spoofing.

3. Execution

Requests are processed by local LLMs (Ollama) in a sandbox. No data leaves your machine except the final result.

4. Solidarity

Credits are social, not financial. Contributing compute builds your reputation in the decentralized ledger.

ASIP is an open-source initiative for agent rights.
